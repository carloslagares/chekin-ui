# Spacing

Chekin uses an **8-point base** with two half-steps. Each value is exported
as `--chekin-space-<n>` and a matching Tailwind utility
(`gap-chekin-2`, `p-chekin-3`, etc.).

<SpacingRuler rows={[
  { token: 'space-0-5', px: 4,  use: 'Hairline gap — icon-to-badge, chip internals.' },

## Rules

- **Gaps before paddings.** Prefer `gap-chekin-2` on a flex container over
  padding-right on every child.
- **Don't invent in-between values.** If `24` feels too large and `16`
  too small, re-evaluate the layout. Ad-hoc `20px` and `28px` are what
  produce inconsistent screens.
- **Mobile tweaks are OK** — use responsive variants (`md:gap-chekin-3`)
  rather than custom paddings.

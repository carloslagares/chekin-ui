# Shadows

Chekin shadows are **blue-tinted** — never pure black — which keeps the
product in the brand's atmosphere even at surface level. Four canonical
elevations.

## Rules

- **Shadows are never pure black** (`rgba(0,0,0, …)`). Use the blue-tinted
  tokens so cards feel cool and cohesive instead of heavy.
- **Hover is not a shadow bump on buttons.** Buttons get an overlay
  (`white/10` or `black/10`). Only *interactive cards* (entire clickable
  surfaces) elevate on hover.
- **Focus ring is non-negotiable.** Every focusable element gets
  `shadow-chekin-focus` on `:focus-visible`. Radix Primitives give this
  for free; custom buttons must add it explicitly.

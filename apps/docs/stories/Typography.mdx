# Typography

The primary typeface is **Montserrat** (400, 500, 600, 700, 800).

- `Inter` is used for the 64px section display on the marketing site only.
- `Poppins` is used inside code / data cells and inside the Guest App only.

All numbers use tabular variants (`font-variant-numeric: tabular-nums`) so
column widths don't jump as values change.

## Marketing scale

Section headers can go up to 64px (Inter, 700) for hero moments. On
Dashboard, the max is 70px H1 and is reserved for the welcome hero —
every other page uses 26–32px for its h1.

## Rules

- Product headings use **Navy** `#161643`. Marketing headings use **Dark** `#19194B`. Never pure black.
- Line-height tightens as size grows (70/84 ≈ 1.2, 14/20 ≈ 1.43).
- Avoid italics — never used in product or marketing.
- Avoid underlines on non-link text.
